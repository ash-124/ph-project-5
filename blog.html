What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element instead of adding listeners to each child element. This works by leveraging **event bubbling**, where events triggered on a child element propagate up to its parent.
Key benefits:

  • Improves performance by reducing the number of event listeners.
  • Handles dynamic elements** efficiently without needing to reattach listeners.
  • Simplifies code maintenance** by centralizing event handling.
